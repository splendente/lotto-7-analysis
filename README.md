# ロト7 分析ツール (Lotto 7 Analysis)

過去のロト7の抽せん結果データを分析し、様々な視点から番号の出現傾向を可視化するWebアプリケーションです。

## 主な機能

- **ヒートマップ**
  過去の抽せん結果における出現頻度を色の濃淡で表します。色が濃いほどよく出ている番号です。
- **コールドナンバー**
  各番号が最後に当せんしてからの空き回数（出ていない期間）を表示します。色が濃いほど長い間出ていない（そろそろ出そうな）番号です。
- **連続出現番号**
  直近の抽せんで連続して当せんしている回数を表示します。色が濃いほど現在連続して出現している番号です。
- **相関性**
  任意の基準番号を選択した際に、その番号と一緒に出やすい他の番号を表示します。色が濃いほど同時に出やすい番号です。
- **ダークモード/ライトモード対応**
  UI右上からテーマの切り替えが可能です。

## 技術スタック

- **フレームワーク**: [Next.js](https://nextjs.org/) (App Router)
- **UI/スタイリング**: [React](https://react.dev/) 19, [Tailwind CSS v4](https://tailwindcss.com/)
- **アイコン**: [Lucide React](https://lucide.dev/guide/packages/lucide-react)
- **リンティング・フォーマット**: [Biome](https://biomejs.dev/)
- **パッケージマネージャ**: pnpm

## 開発環境のセットアップ

1. リポジトリをクローンまたはダウンロードします。
2. 依存パッケージをインストールします。
   ```bash
   pnpm install
   ```
3. 開発サーバーを起動します。
   ```bash
   pnpm dev
   ```
4. ブラウザで [http://localhost:3000](http://localhost:3000) を開き、アプリケ―ションを確認します。

## コマンド一覧

- `pnpm dev`: 開発サーバーの起動
- `pnpm build`: 本番ビルドの生成
- `pnpm start`: 本番サーバーの起動
- `pnpm lint`: BiomeによるLintチェック
- `pnpm format`: Biomeによるフォーマット実行

## データ管理

分析に使用されるベースデータは `src/data/results.json` に格納されています。このJSONファイルには、過去の抽せん回号や本数字などの情報が記録されており、UIの表示に必要な各種統計を動的に計算しています。
